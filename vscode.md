# Settin up vscode
- install ebusd https://marketplace.visualstudio.com/items?itemName=ebusd.ebus-notebook
- install truetype https://marketplace.visualstudio.com/items?itemName=typespec.typespec-vscode
- Install types
    npm i @ebusd/ebus-typespec
- Upgrade npm
    npm install -g npm@11.14.0


- Add --enabledefine to ebusd to be able to interact with vscode
